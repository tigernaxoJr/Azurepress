產品服務

Jform 萬用表單系統

Jform 能讓 IT 人員提供 HTML 及經過簡單設定就能產生表單讓使用者填寫，不需要為每一張表單撰寫後端程式、設計資料表，從而已倍數減少相當百分比的開發、維護人力，採用這個系統的公司能夠有效減少人員花費。

主要功能

✓
系統可設定在使用者提交表單時透過 webhook 通知其他系統實現系統介接
✓
可以直接針對填寫完成的表單拿到 JSON 資料方便作物件關聯對應以分析
✓
可透過簡易設定就能讓表單存取其他資料庫的資料並自動回填表格，不需要撰寫 API 和 AJAX
✓
可透過簡易設定實現在表單內新增不同的明細的檔而不需要再為各明細檔撰寫程式和開立表格
✓
內建 Anti CSRF Token 保護
